# 金融数据分析与压力情景分析工具

## 项目概述

本项目是一个基于Python的金融数据分析工具，集成了多种金融分析功能，包括基础数据分析和压力情景分析。工具可以从多个数据源获取金融数据，计算关键金融指标，并基于大语言模型进行压力情景分析，提供风险评估和对冲建议。

## 功能特性

### 1. 多金融工具数据分析
- 支持股票、加密货币、指数、外汇等多种金融工具
- 使用Alpha Vantage API获取实时高质量金融数据
- 自动计算关键金融指标：
  - 持仓占比
  - 盈亏
  - 日波动率
  - 在险价值(Value at Risk)
  - 贝塔系数

### 2. 压力情景分
- 基于大语言模型分析金融事件类型和影响因子
- 评估事件对投资组合的影响
- 提供个性化对冲建议
- 集成多模型(douabo-thinking,deepseek-v3)分析结果，提高准确性

## 代码结构

### financial_data_analyzer_alpha_vantage.py
核心分析器类文件，包含以下主要方法：
- `get_past_three_months_data(symbol, max_retries)`: 获取过去三个月的交易数据，支持重试机制,
- `export_to_excel(data, symbol, filename)`: 将数据导出到Excel文件,最后以表格的形式输出获取到的三个月的交易记录
- `calculate_position_ratio(data, position_size, total_assets)`: 计算持仓占比
- `calculate_pnl(data, position_size, buy_price)`: 计算盈亏
- `calculate_daily_volatility(data)`: 计算日波动率（使用最近30个交易日的数据）
- `calculate_value_at_risk(data, confidence_level)`: 计算在险价值
- `calculate_beta(data, market_symbol)`: 计算贝塔系数

### multi_stock_analyzer.py
多股票分析工具，可以分析多个股票并计算它们在投资组合中的占比。程序会提示用户输入多个股票基本的信息,无需用户自己计算在险价值,盈亏等.

### stress_scenario_analyzer.py
压力情景分析模块，可以分析金融事件对投资组合的影响。主要功能包括：
- 调用大语言模型分析事件类型和严重程度
- 评估事件对投资组合的影响
- 提供对冲建议
- 集成多模型分析结果

### run_stress_analysis.py
压力情景分析的运行脚本，用于读取多股票分析结果并进行压力测试。

## 算法简介

### 金融数据分析算法
1. **持仓占比计算**：(买入价格 × 持仓量) / 总资产
2. **盈亏计算**：持仓量 × (当前价格 - 买入价格)
3. **日波动率计算**：使用过去一个月数据计算每日收益率的标准差
4. **在险价值(Value at Risk)**：使用过去一年数据，采用历史模拟法计算95%置信水平下的在险价值
5. **贝塔系数计算**：β = Cov(Ri, Rm) / Var(Rm)，其中使用SPY作为市场基准

### 压力情景分析算法
1. **事件分类与影响因子识别**：使用大语言模型分析输入的金融事件，识别事件类型和影响因子（如石油价格、风险情绪、美元避险需求等）
2. **集成学习**：调用多个大语言模型分析同一事件，通过集成算法综合各模型输出，提高分析准确性
3. **投资组合影响评估**：基于事件影响因子和投资组合持仓特性，计算各资产受影响程度
4. **对冲策略生成**：根据事件严重程度和影响因子方向，生成个性化对冲建议

## 算法优缺点

### 优点
1. **多模型集成**：通过集成多个大语言模型的分析结果，提高分析准确性和鲁棒性
2. **实时性**：基于最新市场数据和事件信息进行分析
3. **个性化**：根据用户具体持仓情况提供定制化分析和建议
4. **全面性**：涵盖从数据获取、指标计算到风险分析的完整流程
5. **可扩展性**：模块化设计，易于添加新的分析功能和金融工具

### 缺点
1. **依赖API**：需要Alpha Vantage API密钥，免费版有一定限制
2. **模型依赖**：压力情景分析依赖大语言模型，可能存在模型偏差
3. **简化假设**：部分计算采用了简化模型，可能与实际市场存在一定偏差
4. **数据延迟**：API数据可能存在一定延迟，影响分析时效性

## 为什么选择这些算法

1. **金融数据分析算法**：
   - 选择历史模拟法计算VaR是因为它不依赖于特定的概率分布假设，更符合实际市场情况
   - 使用标准差计算波动率是业界通用方法，便于与其他系统对比
   - 贝塔系数计算使用市场基准，能够有效衡量相对风险

2. **压力情景分析算法**：
   - 选择大语言模型是因为其具备强大的自然语言理解和推理能力，能够准确分析复杂的金融事件
   - 采用集成学习方法是为了降低单一模型的偏差和方差，提高分析结果的稳定性
   - 基于影响因子的分析方法能够更细致地评估事件对不同资产的影响

## 运行指南

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置API密钥
项目使用Alpha Vantage API获取金融数据。您可以通过以下方式配置API密钥：
1. 设置环境变量`ALPHA_VANTAGE_API_KEY`
2. 在代码中直接指定API密钥

获取免费API密钥：https://www.alphavantage.co/support/#api-key

### 运行多股票分析
```bash
python multi_stock_analyzer.py
```
程序会提示输入多个股票的信息，计算每个股票的指标，并汇总所有股票的持仓占比。

### 运行压力情景分析
```bash
python run_stress_analysis.py
```
程序会读取金融分析结果，提示输入金融事件描述，并提供风险评估和对冲建议。

## API限制说明

Alpha Vantage免费版API限制：
- 每分钟最多5次请求
- 每天最多500次请求

如果遇到限制，程序会自动等待并重试。

## 错误处理

程序包含了完善的错误处理机制：
- 网络异常处理
- 数据为空检查
- 输入验证
- API访问限制处理
- 文件导出异常处理

当遇到问题时，程序会提供清晰的错误信息和解决建议。
